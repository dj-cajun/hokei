## 2. 구매자 약관 요지 (표시 위치: `/store/2d-sketch-cafe/coupon/terms`)

1. 쿠폰은 **선결제 권리**이며, 결제금은 **업소 계좌/현금**으로 업소에 직접 지불합니다.
2. 쿠폰 QR은 발급 후 **3분간** 유효하며, 1회 사용 후 재사용 불가합니다.
3. 환불·취소는 **업소 정책** 및 베트남 소비자법에 따릅니다.
4. 호케이는 결제 중개가 아닌 **정보·쿠폰 발급 플랫폼**입니다.
5. 분쟁 시 업소와 구매자 1차 협의, 필요 시 호케이 쪽지(`/messages`)로 중재 요청 가능합니다.

## 3. 제휴 업소 계약 요지

1. 업소는 메뉴 가격·품질·위생에 책임집니다.
2. 플랫폼 수수료: **교환 1건당** — `commissionPercent`(%) 우선, 없으면 `commissionFixed`(₫).
3. 수수료는 **주 1회** K BROTHERS 계좌로 송금합니다.
4. 업소 계좌·VietQR 정보의 정확성은 업소 책임입니다.
5. 직원 PIN·일 마감 절차를 준수합니다.

## 4. 개인정보

- 구매: 호케이 회원 ID → coupon API User 매핑 (최소 정보)
- Partner: `ownerId` · 직원 이름 (PIN 해시만 저장)
- 감사: `CouponAuditLog` (교환·현금·마감)
