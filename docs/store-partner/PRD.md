# PRD — 업소 제휴 · 배너 · 모바일 LP

**버전:** 0.1  
**작성:** 2026-06-27  
**상태:** 기획 확정 → Phase 1 개발 대기

---

## 1. 배경·목적

호치민 한인 업소 사장님에게 **「카톡 링크 하나로 끝나는 모바일 가게 페이지 + 호케이 배너 노출」** 을 유료로 제공한다.

- **문제:** 카톡방 글은 묻히고, 자체 웹·앱은 비용·관리 부담
- **기회:** 호케이는 교민 **뉴스·생활정보 트래픽** 보유, `/promo` 업소 데이터 일부 존재
- **목표:** 월 구독형 **제휴 수익** + `ads@hokei.vn` 영업 파이프라인

---

## 2. 타깃 사용자

| 페르소나 | 니즈 |
|----------|------|
| **교민 이용자** | 맛집·미용·병원 정보를 **빠르게** (카톡·전화·지도) |
| **업소 사장님** | 한인 고객 유입, **관리 쉬운** 홍보 페이지 |
| **호케이 운영자** | 등록·배너·만료 관리, 클릭 통계(Phase 2) |

---

## 3. 제품 범위

### Must (Phase 1 — MVP)

- [ ] **모바일 LP** `/store/[slug]` — 히어로, 카톡·전화·지도 CTA, 영업시간, 한 줄 소개
- [ ] **제휴 허브** `/partners` — 등록 업소 목록 + `ads@` 문의 CTA
- [ ] **관리자** — 업소 CRUD (이름, slug, 카톡, 전화, 지도 URL, 썸네일, 플랜, 노출 on/off)
- [ ] **홈 하단 배너 슬롯 1개** — 활성 `PartnerBanner` 수동 연결
- [ ] **SEO** — LP `metadata`, JSON-LD `LocalBusiness` (선택)

### Should (Phase 2)

- [ ] 다중 배너 슬롯 (홈·뉴스·/promo)
- [ ] 클릭·조회 통계 (`PartnerEvent`)
- [ ] `/promo/timeline/[store]` ↔ `/store/[slug]` 상호 링크
- [ ] 만료일·플랜별 자동 비노출

### Won't (코드 범위 밖 · 미구현 예정)

- **PG·자동 결제** — 수금은 `ads@` 수동 계약·이체만 (앱 내 결제 없음)
- 사장님 셀프 서비스 대시보드 (Phase 3 검토)
- 다국어(베트남어) LP

---

## 4. 요금·영업 (참고, **코드·자동 결제 없음**)

| 플랜 | 포함 | 가격 가이드 |
|------|------|-------------|
| Basic | LP + /partners 목록 | $30~50/월 |
| Standard | + 홈 하단 배너 | $80~120/월 |
| Premium | + 홈 상단·뉴스 사이 | $150~250/월 |

첫 **3~5곳** 파일럿: 1~2개월 무료 → 사례 확보.  
**청구:** 이메일·카톡 합의 후 **수동 이체** — Stripe/PG 연동 없음.

## 6. 비기능 요구

- 모바일 퍼스트 (360px~), 터치 CTA min 44px
- 카톡 링크 `isValidKakaoLink` 검증 (기존 lib 재사용)
- 관리자만 등록·수정 (`requireAdmin`)
- 공개 LP는 `status=PUBLISHED` && `expiresAt` null or future

---

## 7. 기존 기능과 관계

- **`/promo`**, `Post.storeName` — 병행. Phase 2에서 slug 연결
- **`ads@hokei.vn`** — 영업·문의 (이미 운영)
- **AdSense** — 프로그래매틱 광고, 제휴 배너와 **슬롯 분리**
